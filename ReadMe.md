# 💫 About Me:
Hi, I'm Edgar.<br><br>I'm a self-taught programmer focused on performance, systems, and security. I don't limit myself to a single area: I work on everything from malware and low-level system tools to complete web platforms and games, even projects in the electronics field. Basically, if it involves code, I make it happen.<br><br>🛠️ What I use:<br><br>Low-Level & Systems: C, C++, Zig, Assembly, Go (Performance and binary exploration)<br><br>Scripts & Automation: Python, Bash, Powershell<br><br>Web Full-Stack: JavaScript, TypeScript, React, Node.js, PHP, HTML/CSS<br><br>Mobile & Enterprise: Java, C#<br><br>Databases: MySQL, sqlite<br><br>💻 What I develop:<br><br>Malware & Security: Security research and creation of offensive tools.<br><br>Tools: Useful scripts and automation software for desktops.<br><br>Web: Complete websites with cloud servers, APIs, and databases.<br><br>Games: Game projects and engines. <br><br>Basically, I do a little bit of everything, with no exception.
